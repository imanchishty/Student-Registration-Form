Project Title:
Student Registration Form

Student Name:
Iman Chishty

Objective:
To design a  HTML registration form that collects a student's personal, academic, additional, and account information in an organized, sectioned layout.

Features Implemented:
1- Form divided into 4 sections: Personal, Academic, Additional, and Account Information
2- Next and Back buttons
3- Skills selection using checkboxes
4- Profile picture upload with image input field
5- Short bio input using a textarea
6- Username and password fields with password confirmation
7- Reset and Submit buttons
8- Required field validation
9- Icons integrated using Font Awesome and Unicons

HTML Elements Used:
<form>, <fieldset>, <legend>
<div>, <span>, <label>
<input> — types: text, email, number, date, radio, checkbox, file, password
<textarea>
<button> — types: button, reset, submit
<link> — for external icon libraries (Font Awesome, Unicons)
<i> — for displaying icons
<p>, <header>

How to Run the Project:
In VScode, press on "Go-Live" button at the bottom right corner. The project will run.
